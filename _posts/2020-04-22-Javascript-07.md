## Object

```js
const harry = {
  name: "harry",
  age: 27,
  gender: "Male",
  isHandsome: true,
};
console.log(harry);
console.log(harry.age);
```

`Object`는 배열과 다르다.

`Object` key,value로 만들 수 있으며, `.`으로 key에 접근할 수 있다.
