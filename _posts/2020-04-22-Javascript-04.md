## let, const, var

변수의 값이 상수가 되기를 원한다면 `const`를 사용하자.

```js
const a = 1000
a = 10 // Error!
console.log(a)
```

`var`을 써보자.

`var`은 `let`과 마찬가지로 값을 바꿀 수 있다.

```js
var a = 1000
a = 5
console.log(a)
// 5
```

그러나 `var`은 오래되었고 가급적이면 `let`을 쓰도록 하자.
